Detailed Description

An implementation of Renderer to draw meshes using OpenGL calls from the OpenGL ES 2 collection. The draw function in this will work in desktop OpenGL or in OpenGL ES 2.

Set Sprite::renderer to an instance of RendererGL2 to and call init(). Then all meshes will draw using it.
