Detailed Description

Renderer is an abstract base-class whose virtual methods define how a Mesh shall be drawn. Subclasses of Renderer represent a scheme for drawing a mesh in a particular graphics library.

To implement a renderer, implement init() and drawMesh(). Whichever renderer Sprite::renderer is set to gets used to draw all sprite graphics.
